# Model Checkpoints

Place the trained checkpoint files in this folder before running the Streamlit app.

Required files:

- `convnext_tiny_cleaned_best_weighted_f1.pth`
- `resnet18_cleaned_best_weighted_f1.pth`
- `resnet18_finetune_best.pth`
- `clip_finetune_best.pth`
- `clip_linear_probe_best.pth`

These files are not committed to GitHub because they are large model artifacts.

Checkpoint download link:
